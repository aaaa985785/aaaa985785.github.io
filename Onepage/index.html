    รับยอดเสียในค่าย JOKER สูงสุดถึง 5％

    1. โปรโมชั่นนี้ สำหรับการเล่น เกมสล็อตและยิงปลา ในค่าย JOKER เท่านั้น (นับเฉพาะยอดเสีย)
    2. โบนัส=ยอดเสีย-ยอดที่รับโปรโมขั่น。
    EX：ยอดฝากที่เสียไป 10,000 บาท ลบกับ ยอดฝากที่รับโปรโมชั่น1,000บาท เท่ากับ โบนัสที่จะได้รับ（10,000-1,000）＝9,000，โบนัสที่จะได้รับ 180 บาท
    3. ช่วงเวลาคำนวณแต่ละวัน 00:00:00 ~ 23:59:59
    4. โบนัสไม่จำเป็นต้องแจ้งสมัคร ระบบจะแจกอัตโนมัติ ก่อนเวลา 18：00 น. ของวันจันทร์
    5. ก่อนทำการถอน ต้องทำเทิร์น 1 เท่า โบนัสนี้ เล่นได้ทุกค่ายเกมส์

    วันพุธที่สดใส กับโปร ยิงปลา

    1. สมาชิกสามารถทำการสมัครได้ทุกวันพุธ
    2. โปรดังกล่าวจำกัดเกมยิงปลาเท่านั้น JILI，JDB，SG，AW，CQ9，KA，RSG，FC，GMT，PS (หากเล่นเกมอื่นจะไม่คำนวณเทิร์นโอเวอร์รวม โบนัสและกำไรที่ได้ระบบจะหักทั้งหมดและไม่สามารถขอคืน)
    3. ไม่สามารถใช้ร่วมกับโปรอื่นได้ (ไม่นับรวมในการคืนโบนัสยอดเสีย)
    4. วิธีสมัครโปรโมชั่น : เข้าหน้าเว็บ>>ไปที่ศูนย์สมาชิก>>โปรโมชั่น>>ใบสมัคร>>กดสมัครโปรที่ต้องการ

    6. กรณีที่ออกโปรไม่ได้ กรุณารอประมาณ 15-20 นาที หากเลยเวลาดังกล่าวแล้วยังไม่สามารถออกโปร กรุณาติดต่อแอดมินไลน์ LINE ID : @SBFPLAY (มี@ นำหน้าด้วยนะคะ)
